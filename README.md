# Pixlo Holidays

A modern, responsive travel landing page for Pixlo Holidays, built as a single-file static website with Tailwind CSS via CDN. The site showcases signature itineraries for Himachal, Goa, and Kerala, with a premium visual style, travel-focused content, and direct WhatsApp and email contact options.

## Features
- Responsive one-page layout for desktop and mobile
- Elegant hero section with strong travel branding
- Detailed itinerary cards with interactive accordion-style day plans
- Clear WhatsApp and email contact calls to action
- Ready to publish on GitHub Pages from the root folder

## SEO Optimizations
- **Enhanced meta tags**: Title, description, keywords, author, robots
- **Canonical URL**: Prevents duplicate content issues
- **Open Graph tags**: Rich social media sharing previews
- **Twitter Card**: Optimized for Twitter sharing
- **JSON-LD Structured Data**: Schema.org markup for TravelAgency and TouristTrip
- **Theme color**: Mobile browser theme color for better branding

## Performance Optimizations
- **Preconnect hints**: Early connection to Google Fonts
- **Lazy loading**: Images load only when needed
- **Async decoding**: Non-blocking image decoding
- **Theme color meta**: Faster mobile browser rendering
- **Mobile web app capable**: Installable PWA support

## Accessibility Features
- **Skip to content link**: Keyboard navigation support
- **ARIA labels**: Navigation and interactive elements
- **Semantic HTML**: Proper use of main, nav, article, section
- **Role attributes**: Landmark roles for screen readers
- **Alt text**: Descriptive image alt attributes

## Deployment
Upload the repository contents to GitHub Pages or use the repository's main branch as the published source.
